What is HSGS Code Camp?

HSGS Code Camp is a project where HSGS's students go to 6 secondary schools in Hanoi to teach students there about C++ programming language. Secondary students will be taught the fundamental of C++ and competitive programming. The main platform for practicing is Codefun.vn - a product of HSGS students.
